# MidNight-Bot
## The official bot of MidNight Discord server.
---
## Features
### Slash Commands
- Ban/Mute/Kick
- Unban/Unmute
- Purge
- User info / Server info
- Status / help
- User history
- Create embed

### Other
- User/Staff permissions
- Ticket unban
- Logs - sanctiuni / notificari - unban
- Kick system
- Security
- Database
- Welcome message
- Setup Commands
